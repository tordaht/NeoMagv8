🧬 NeoMag v8.2.1 (v821) - AI Bakteri Simülasyonu

Sürüm: v821 | Tarih: 2025-06-06 | Status: Stable

Tech Stack: TensorFlow.js + Custom ML + Real-time Learning

📋 Program Genel Yapısı

NeoMag v821, browser tabanlı yapay zeka destekli bakteri simülasyonudur. Gerçek makine öğrenmesi algoritmaları ile bakterilerin öğrenme, büyüme ve sosyal etkileşim süreçlerini simüle eder.

🏗️ Temel Mimari:

  • Frontend: HTML5 + TailwindCSS + Vanilla JavaScript
  • ML Engine: TensorFlow.js v4.15.0 (Browser Native)
  • Learning System: Custom Rule-based + Neural Networks
  • Communication: Stand-alone (No External APIs)

🧠 Core AI Sistemi

1. Neural Network Models

wordEmbeddingModel: Sequential Model - Input: [50] (word features) - Hidden: [32] relu + dropout(0.2) + [16] relu - Output: [8] softmax behaviorPredictionModel: Sequential Model - Input: [12] (bacteria features) - Hidden: [24] relu + dropout(0.1) + [12] relu - Output: [6] softmax (behavior classes)

2. Word Learning System

  • Dynamic Tokenizer: Real-time kelime tokenizasyonu
  • User Input Learning: Sohbetten yeni kelimeler çıkarma
  • Word Generation: Mevcut kelimelerden yeni kombinasyonlar
  • Cross-Bacteria Sharing: Bakteriler arası kelime paylaşımı

3. Behavior Prediction Engine

12-feature vector ile bakteri davranışı tahmini:

Features: [age, size, consciousness, optimism, sociability, vocabulary_size, memory_count, growth_rate, hunger, energy, social_interactions, recent_learning] Outputs: creative | social | learning | basic

🦠 Bakteri Sınıfı Yapısı

class BacteriaEntity

Properties: - biological_state: {size, age, growth_rate, division_potential} - personality_traits: {optimism, sociability} - consciousness_level: 0-5 arası bilinç seviyesi - language_stage: 0-5 arası dil gelişim aşaması - vocabulary: Set() - dinamik kelime haznesi - memory_bank: [] - öğrenme ve deneyim kayıtları - conversation_history: [] - sohbet geçmişi - position: {x, y, vx, vy} - hareket koordinatları Methods: - updateDaily(): Günlük büyüme ve gelişim - lookForFood(): Yemek arama algoritması - eatFood(): Beslenme ve büyüme mekanizması - move(): Fiziksel hareket simulasyonu

💬 Chat Sistemi

3 Chat Modu:

  • Individual Chat: Tek bakteriyle birebir sohbet
  • Group Chat: Tüm bakterilerle aynı anda iletişim
  • Bacteria Conversation: Bakterilerin kendi aralarında konuşması

generateBacteriaResponse()

ML-enhanced response generation:

1. User input'tan kelime öğrenme (learnFromUserInput) 2. Behavior prediction (predictBehavior) 3. Consciousness level'a göre template seçimi 4. Behavioral mode'a göre response customization 5. Yeni kelime üretme (generateNewWord)

Cross-Bacteria Learning

bacteriaLearnFromEachOther(): - Vocabulary sharing (30% chance per word) - Social influence on personality traits - Consciousness boost from interaction - Memory bank updates groupLearningSession(): - Collective vocabulary pooling - 20% learning probability per word - Group social influence effects
